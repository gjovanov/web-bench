# Install deps
`npm i`

# Start dotnet core
```bash
cd dotnet
dotnet build
dotnet run -p .\dotnet.csproj --configuration Release
```

# Start uWebSockets.js
```bash
cd uws
npm i
npm run start
```

# Start Fastify.js
```bash
cd fastify
npm i
npm run start
```

# Start Express.js
```bash
cd express
npm i
npm run start
```

# Run benchmark tests
```bash
cd test
npm i
npm run test
```
